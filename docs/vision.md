# Visão do Projeto

## Problema
Usuários que buscam imóveis precisam navegar por empreendimentos, comparar opções e ver detalhes relevantes como localização, faixa de preço e unidades/plantas disponíveis.

## Proposta
Construir uma plataforma web simplificada onde usuários possam explorar empreendimentos imobiliários cadastrados em um banco de dados.

## Público-alvo
Pessoas interessadas em buscar imóveis em uma cidade específica.

## Escopo do MVP
A primeira versão do sistema deverá permitir:
- Listar os empreendimentos (`Developments`)
- Filtrar empreendimentos por cidade (`City`)
- Visualizar a página de detalhes de um empreendimento
- Visualizar as unidades e plantas (`Units`) disponíveis no empreendimento
- Prover um Call-to-Action (CTA) para contato com o corretor (ex: botão do WhatsApp)

## Fora de Escopo (Por enquanto)
- CMS/blog
- Chatbot
- Integração com sistemas de construtoras
- Sistema interno de atendimento (CRM)
- Autenticação de usuários
- Painel administrativo (Admin panel)
